* Install

Requirements:
- Java 8, Scala 2.12.12
- [[https://www.scala-sbt.org/][sbt]] v1.4.3 or higher

You can then add Proter as a dependency from Maven.

e.g. in sbt: 
#+BEGIN_SRC scala
libraryDependencies += "com.workflowfm" %% "proter" % "0.6.1"
#+END_SRC
